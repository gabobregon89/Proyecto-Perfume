# Web de perfumes

Sitio estatico simple para mostrar perfumes, precios y recibir consultas por WhatsApp.

## Qué debes cambiar primero

- Edita `script.js` y reemplaza `5491100000000` por tu número real de WhatsApp en formato internacional.
- Actualiza el arreglo `perfumes` con tus productos, precios y descripciones.

## Cómo publicar rápido

- Puedes subir estos archivos a GitHub y desplegarlos con GitHub Pages, Cloudflare Pages o Netlify.
- Como es una web estática, no necesitas servidor ni base de datos.

## Estructura

- `index.html`: estructura principal.
- `styles.css`: diseño visual y responsive.
- `script.js`: catálogo y enlaces de WhatsApp.
